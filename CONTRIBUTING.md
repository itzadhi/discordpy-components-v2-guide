# Contributing

Thank you for helping improve this Discord learning ecosystem.

## Standards

- Teach the concept in human language before code.
- Preserve the 12-section lesson structure.
- Use modern Discord API practices only.
- Mark outdated methods with explicit warnings and migration guidance.
- Keep examples runnable, typed, and safe.
- Do not commit tokens, guild IDs, private logs, or generated secrets.

## Local Verification

```bash
npm install
npm run verify
```
